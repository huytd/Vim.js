# Vim.0.1.js
Vim for web textarea and input field. To improve writing experience.

**This project is in development!**

# supported browser

* Chrome  v39
* Firefox  v34
* Safari

# supported command

* esc : switch mode to general
* j | k | h | l : move cursor
* v | V : switch mode to visual
* a | A
* i | I
* o | O
* r
* y | yy
* p
* x
* 0
* $
